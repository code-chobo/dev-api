# 자바 초보 스터디 api 개발
- [남궁성의 코드 초보 스터디](https://cafe.naver.com/javachobostudy) 카페의 플랫폼 독립을 위한 프로젝트입니다.

## 환경
- 기술 스택 ([pom.xml](https://github.com/code-chobo/dev-api/blob/develop/pom.xml))

    - Spring Boot
    - Spring Data Jpa
    - QueryDSL
    - Spring Rest Docs
    - Spring Security
    - MySQL
    - JUnit5
    - Jwt
- 도구
    - Git / Github
    - Jira
    - IntelliJ IDEA
